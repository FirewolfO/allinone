由于Shell脚本是从头开始执行的，所以Shell规定，函数必须在调用前定义

# 函数的定义

```powershell
function_name () 
{ 
	statement1 
	statement2 
	... 
	statement n 
w}
```

或者

```powershell
function function_name () 
{ 
	statement1 
	statement2 
	... 
	statement n 
w}
```



# 函数的调用

```powershell
function_name param1 param2 …
```
