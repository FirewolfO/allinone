# 类加载子系统概述

- 类加载子系统负责从文件系统或者是网络系统中加载class文件；

- class文件在开头有文件标识，也就是魔数；

- Class Loader负责加载类文件，至于是否能够运行，则由执行引擎决定；

- 加载的信息存放于一块称为方法区的内存空间。除了类的信息之外，方法区中还会存放运行时的常量信息，可能包括字符串字面量和数字常量。
